# MVP Scope

## In scope for V1 planning

The first implementation should center on owner coordination and keep communication attached to work:

- tasks
- comments on work items
- shared goals
- goal progress updates
- owner collaboration
- a simple recent activity view

## Out of scope for V1 implementation

These items are intentionally deferred:

- direct chat or standalone messaging
- full inventory management
- purchasing automation
- invoicing workflows
- vendor ordering flows

## Explicit later phases

The broader product still includes:

- inventory
- purchasing
- vendor management
- invoice handling

V1 should not block those modules, but it should not implement them yet.
